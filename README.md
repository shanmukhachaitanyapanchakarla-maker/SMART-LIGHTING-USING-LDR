# SMART-LIGHTING-USING-LDR

A Smart Lighting System is an automated lighting solution  that controls a light source based on the surrounding light intensity. This project uses a Light Dependent Resistor (LDR) and an Arduino UNO to automatically switch an LED ON during low-light conditions and  OFF when sufficient light is available.

## 📖 Project Overview

The Smart Lighting System Using LDR is an embedded systems project designed in tinkercad which automatically control lighting based on ambient light conditions. The system utilizes a Light Dependent Resistor (LDR) to sense environmental light intensity and intelligently switch a lighting device ON or OFF. This automation helps reduce energy consumption and improves lighting efficiency.

## ⚙️ Working Principle

The system continuously monitors ambient light intensity using an LDR sensor. The sensor's resistance changes according to the amount of light falling on it.

- During daylight conditions, the system keeps the light source OFF.
- During low-light or dark conditions, the system automatically turns the light source ON.

This enables automatic lighting without human intervention.
